Nível Intermédio

Após uma explicação sobre os conceitos básicos dos Warframes, iremos agora explicar o que consiste cada estilo de playstyle.

Agilidade

Os jogadores que normalmente jogam segundo este playstyle tendem a não depender muito sobre as qualidades de resistência do Warframe, nem das suas habilidades.

Por norma este estilo é praticado em todos os playstyles no entanto o uso dele é acentuado neste, uma vez que os Warframes são feitos de forma a tirar o máximo desse proveito.
Ou seja, são colocados mods que benefeciam essencialmente a velocidade do Warframe.

Warframes comuns para este playstyle

Volt

Gauss

Poderá aprender mais como buildar estes Warframes em especifico seguindo para o nivel complexo.
Ambos Warframes têm em comum é o facto de que ambos usam uma habilidade que lhes dá mais velocidade.
Exemplos vão ser dados na secção seguinte uma vez que não existe uma build que se possa usar em todos e que ao mesmo tempo seja viável.

Tank

Para este tipo de playstyle os jogadores tendem a fazer os seus Warframes resistentes ao dano.

Isto é, já estão a considerar receber dano e como tal fazem para que o dano recebido seja o minimo possivel.

Warframes comuns para este playstyle

Rhino

Inaros

Como na secção anterior estes Warframes têm em comum é uma habilidade que lhes dão um aumento de resistências.
Exemplos vão ser dados na secção seguinte uma vez que não existe uma build que se possa usar em todos e que ao mesmo tempo seja viável.
